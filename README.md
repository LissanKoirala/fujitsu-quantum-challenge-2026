# Fujitsu Quantum Challenge 2026

Small team repo for the 2026 challenge (programmers, physicists, logisticians).

## Quick Start
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pytest tests/
```

## Where Things Live
- Code: `src/`
- Notebooks: `notebooks/`
- Tests: `tests/`
- Planning: `project_management/`
- Docs: `docs/`
